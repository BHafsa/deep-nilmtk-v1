class deep_nilmtk.disaggregator.nilm_experiment.NILMExperiment(params)

Bases: Disaggregator

This class defines a NILM experiment. It is compatible with both single and multi-appliance models and offers different advanced features like cross-validation and hyper-parameters optimization during the training phase. The class is independent of the deep model used for load disaggregation.

Note

For a PyTorch model to be compatible with this class, an entry should be added for this model in the config module.
